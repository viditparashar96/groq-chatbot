## TROUBLESHOOTING GUIDE

### Mobile Network Issues
1. **No signal / network not available:**
   - Toggle Airplane mode on and off
   - Restart your phone
   - Check if SIM is properly inserted
   - Verify your plan is active (dial *121#)
   - Check for network outages in your area via the app

2. **Slow data speed:**
   - Check if daily data limit is exhausted
   - Switch between 4G/5G in phone settings
   - Move to an area with better signal
   - Clear app cache and restart browser

3. **Call drops:**
   - Check signal strength
   - Try Wi-Fi calling if available
   - Report persistent issues at 198

### Broadband Issues
1. **Internet not working:**
   - Check if the router power light is on
   - Restart the router (unplug for 30 seconds)
   - Check LAN/Wi-Fi connection on your device
   - Verify bill payment is up to date

2. **Slow speed:**
   - Run a speed test at fast.com
   - Connect via LAN cable to test (eliminates Wi-Fi issues)
   - Check number of connected devices
   - Router placement: keep in a central, elevated location

3. **Wi-Fi connectivity issues:**
   - Restart the router
   - Forget and reconnect to the Wi-Fi network
   - Switch between 2.4 GHz and 5 GHz bands
   - Update router firmware via the Airtel Thanks App

## ADVANCED TROUBLESHOOTING

### SIM Card Issues

**SIM Not Detected:**
1. Power off the phone
2. Remove the SIM card and clean the gold contacts with a dry cloth
3. Re-insert the SIM card firmly
4. Power on and wait 30 seconds
5. If still not detected, try the SIM in another phone to confirm if SIM or phone issue
6. If SIM works in another phone: Phone's SIM tray may be damaged
7. If SIM doesn't work in any phone: Visit Airtel store for SIM replacement (free with ID proof)

**SIM Blocked / PUK Required:**
1. Enter wrong PIN 3 times → SIM gets locked
2. Need PUK code to unlock
3. Get PUK code: Call 121 from another number, or visit Airtel store with ID proof
4. Enter PUK code + new PIN to unlock
5. Warning: 10 wrong PUK attempts permanently blocks the SIM

### Data Not Working

**Step-by-step diagnosis:**
1. Check if mobile data is enabled (Settings > Mobile Data > ON)
2. Check if Airplane mode is OFF
3. Verify APN settings:
   - APN: airtelgprs.com
   - No username or password required
   - MCC: 404 or 405 (depending on circle)
   - MNC: Varies by circle
4. Reset network settings (Settings > Reset > Reset Network Settings)
5. Check if data pack is active: Dial *121# > Data Balance
6. If FUP exhausted: Speed reduced to 64 Kbps, purchase data add-on
7. If all fails: Contact 121 for network-side reset

### VoLTE / Wi-Fi Calling Issues

**Enable VoLTE:**
1. Settings > Mobile Network > Preferred network type > 4G/LTE
2. Settings > Mobile Network > VoLTE calls > Enable
3. Restart phone
4. VoLTE icon should appear in status bar

**Enable Wi-Fi Calling:**
1. Settings > Wi-Fi Calling (or Connections > Wi-Fi Calling)
2. Toggle ON
3. Accept terms and conditions
4. Wi-Fi Calling icon appears when connected to Wi-Fi
5. Calls will automatically route over Wi-Fi when cellular signal is weak

**Supported phones for Wi-Fi Calling:**
- iPhone 6s and later
- Samsung Galaxy S8 and later
- OnePlus 6 and later
- Xiaomi/Redmi Note 7 and later
- Most phones launched after 2019 with VoLTE support

### Bill Payment Issues

**Postpaid Bill Not Received:**
1. Check registered email (including spam folder)
2. Download bill from Airtel Thanks App > My Account > Bills
3. Call 121 to request bill re-send

**Payment Failed But Amount Debited:**
1. Wait 24-48 hours for auto-reversal
2. If not reversed: Take screenshot of bank debit
3. Call 121 with transaction reference number
4. Or raise complaint in Airtel Thanks App > Support > Billing

**Auto-Pay Setup:**
1. Open Airtel Thanks App
2. Go to My Account > Auto-Pay
3. Choose UPI, debit card, or credit card
4. Set up auto-debit for postpaid, broadband, or DTH
5. Get Rs.10-50 cashback on auto-pay activation (promotional)
